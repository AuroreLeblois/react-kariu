# Welcome to react-kariu Library

This library will contain several useful components for your React projects.

- Avatar
- Button
- Input (beta + work-in-progress)

## How to use "react-kariu"

### Instalation

```
npm i react-kariu
```

### Usage

```
import { Button } from 'react-kariu'

<Button
  label="Button"
  onClick={() => {}}
  shape="square"
/>

```

### Documentation Usage

Play and customise your component with the react-kariu documentation. Once your component is ready, go to the Docs section of the component
and click on "show code" for the raw code of your component.
