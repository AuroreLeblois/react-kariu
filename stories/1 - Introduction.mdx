# Welcome to react-kariu Library

This library will contain several useful components for your React projects.

### Available components

- **Avatar** - Display component for user avatars
- **Button** - Customizable interactive button
- **Card** - Container for displaying content
- **CheckBox** - Checkbox component with ripple effect and custom css variables
- **HoverAnimation** - Animation for element hover effects
- **Link** - link to internal or external href
- **SlideAnimation** - Sliding animation for elements
- **Switch** - Switch component with ripple effect and custom css variables
- **ThemeProvider** - Theme provider with features like useTheme, setTheme, calculateShades
- **Ripple** - Ripple effect for elements

### Work-in-progress

- **Input** - Input field for forms (Beta)
- **Calendar** - Interactive calendar component
- **Carousel** - Carousel component with custom css variables from swiper library is still in progress and quite bugged

## How to use "@react-kariu"

### Installation

```
npm i @react-kariu

```

### Usage

```
import { Button } from 'react-kariu'

<Button
  label="Button"
  onClick={() => {}}
  shape="square"
/>
```

### Documentation Usage

Play and customize your component with the react-kariu documentation. Once your component is ready, go to the Docs section of the component
and click on "show code" for the raw code of your component.
